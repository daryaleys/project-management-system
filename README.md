# project-management-system

Система управления проектами, позволяющая создавать и редактировать задачи для проектов.

Приложение состоит из:
1. Страницы всех задач
2. Страницы проектов
3. Страницы задач конкретного проекта

## Функционал
1. Просмотр всех задач на странице /tasks
2. Поиск по названию задачи и по исполнителю
3. Фильтр по статусу задачи и по проекту, к которому он привязана
4. Просмотр проектов на странице /boards
4. Просмотр доски проекта на странице /boards/:id
5. Детальный просмотр задачи с помощью модального окна
7. Создание задачи с помощью кнопки в хэдере и с помощью кнопки на странице всех задач
8. Редактирование задачи со страницы всех задач и со страницы доски
9. Возможность со страницы всех задач перейти на страницу доски с открытием детальной информации о выбранной задаче

## Стек технологий
`React`, `TypeScript`, `React Router`, `TanStack Query`,  `axios`, `Vite`.

## Запуск приложения
### Запуск сервера
Запуск сервера описан в README в папке server.

### Запуск проекта
1. Установить [NodeJS и NPM](https://nodejs.org). (Версия 20.11)
2. Перейти в папку frontend `cd frontend`.
3. Выполнить `npm i`.
4. Выполнить `npm run dev` для запуска приложения на https://localhost:5173.
Для production-сборки используйте `npm run build`.
